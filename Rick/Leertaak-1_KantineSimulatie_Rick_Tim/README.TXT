PROJECT TITLE: Kantine Simulatie
PURPOSE OF PROJECT: Simuleert een aangemaakte kantine, waarbij het aantal dagen ingevult word als int. Elke dag worden er 2 extra klanten toegevoegd waarbij het minimaal aantal klanten 10 is. Elke klant besteld 2 producten.
VERSION or DATE: 19-11-2014
HOW TO START THIS PROJECT: "new KantineSimulatie()", vervolgens in het aangemaakt object de methode "void simuleer(int dagen)" uitvoeren.
AUTHORS: Tim Zijlstra, Anna Woudstra en Rick Wolthuis
